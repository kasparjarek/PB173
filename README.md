# PB173
PB173 - Tematicky zaměřený vývoj aplikací v jazyce C/C++ homeworks

## How to build
```
$ cmake .
$ make
```

## How to use
Run world
```
$ bin/world --area-size 10 10 --green-tanks 2 --red-tanks 2 -p pipe --round-time 10000
```
Run worldclient
```
$ bin/worldclient -p pipe
```
Run tankclient
```
$ bin/tankclient
```
